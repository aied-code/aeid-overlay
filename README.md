# aeid-overlay (아이드 점수 조회 오버레이)

[aeid.gg](https://aeid.gg)의 환산 점수를 게임 화면 위에 오버레이로 즉각 표시해 주는 유틸리티 프로그램입니다. 매번 웹사이트에 접속해 검색할 필요 없이, 게임 플레이 중 자연스럽게 파티원이나 신청자의 스펙을 확인할 수 있습니다.

## 주요 기능 및 작동 방식

* **스마트한 화면 표시**: 평상시에는 화면에 보이지 않아 게임 플레이를 방해하지 않습니다.
* **자동 팝업**:
    * 초월, 원정 창 등에서 특정 캐릭터를 클릭할 때
    * 파티장일 때, 파티 신청자가 생길 때 자동으로 오버레이가 나타납니다.
* **상세 보기 지원**: 오버레이 내 '상세 보기' 버튼을 클릭하면 즉시 해당 캐릭터의 `aied.gg` 페이지로 연결됩니다.
* **자유로운 창 이동**: 상단 바를 마우스로 드래그하여 원하는 위치로 자유롭게 이동할 수 있습니다.

## 조작 방법

* **숨기기**: `ESC` 키를 누르거나 창의 `빨간색 닫기 버튼`을 클릭하면 오버레이가 다시 숨겨집니다.
* **완전 종료**: 작업표시줄에서 해당 프로그램의 아이콘을 우클릭하여 창을 닫으면 완전히 종료됩니다.

## 설치 및 실행 방법

### 1. 필수 프로그램 설치 (NPCAP)
이 프로그램은 네트워크 패킷을 읽기 위해 **반드시 NPCAP이 설치되어 있어야 정상 작동**합니다.
* [NPCAP 다운로드 공식 링크 (https://npcap.com/#download)](https://npcap.com/#download)
* 위 링크에 접속하여 최신 버전을 설치해 주세요.

### 2. 프로그램 설치
1.  본 저장소의 **[Releases]** 탭으로 이동합니다.
2.  가장 최신 버전의 `.msi` 설치 파일을 다운로드합니다.
3.  다운로드한 `.msi` 파일을 더블 클릭하여 설치를 진행합니다.
4.  게임을 먼저 실행한뒤 해당 프로그램을 실행하면 됩니다.

## ⚠️ 주의 사항 및 면책 조항 (필독)

본 프로그램을 사용하기 전에 아래 내용을 반드시 숙지해 주시기 바랍니다.

> **1. 비인가 프로그램 오인 가능성**
> 이 프로그램은 딜 미터기와 유사하게 네트워크 패킷을 감시하는 방식을 사용합니다. 따라서 타인에게 딜 미터기 사용자로 오인받을 가능성이 존재합니다.
> 
> **2. 클라이언트 무결성 보장**
> 이 프로그램은 게임 클라이언트를 직접 조작하거나 메모리를 변조(핵 등)하는 행위는 **일절 하지 않습니다.** 오직 패킷 감시를 통해 '닉네임'과 '서버 정보'만을 읽어옵니다.
> 
> **3. 사용자의 책임**
> 이 프로그램의 사용으로 인해 발생하는 계정 정지 등의 모든 불이익과 책임은 **전적으로 사용자 본인에게 있음**을 알립니다.
